    Vector Search ハンズオンへようこそ¶

    このハンズオンでは、IBM Bob を使って「意味で検索する」機能（Vector Search）を体験します。

    前提条件

    IBM Bob が既にインストールされ、使用できる状況を前提としています。

    Vector Search とは？¶

    Vector Search（ベクトル検索） は、言葉の「意味」を理解して検索する技術です。

    従来の検索との違い¶

    従来のキーワード検索:

    • 「赤いスニーカー」→「赤い」と「スニーカー」という文字が含まれる商品を探す
    • 「赤色のランニングシューズ」は見つからない（文字が違うため）

    Vector Search（意味で検索）:

    • 「赤いスニーカー」→「赤い」「スニーカー」の意味を理解
    • 「赤色のランニングシューズ」も見つかる（意味が似ているため）
    • 「初心者向けカメラ」→「入門用デジタルカメラ」も見つかる

    実際の活用例¶

    • EC サイト: 「似た商品を探す」機能
    • 社内検索: 「この資料に似た文書を探す」
    • カスタマーサポート: 「似た質問を探す」

    IBM Bob とは？¶

    IBM Bob は、AI がコーディングをサポートしてくれる開発ツールです。

    IBM Bob でできること¶

    • 自然な日本語で指示: 「検索結果に画像を表示して」と伝えるだけ
    • コードを自動生成: IBM Bob がコードを書いてくれる
    • コードレビュー: 書いたコードの問題点を指摘してくれる

    従来の開発との違い¶

    従来の開発: プログラミング言語を学ぶ → コードを書く → バグを修正 → テスト（数日〜数週間）

    IBM Bob を使った開発: 日本語で指示 → IBM Bob がコード生成 → 動作確認（数分〜数時間）

    事前準備（15 分）¶

    Vector Search Builder をセットアップして、ハンズオンの準備をします。

    やること:

    1. プロジェクトフォルダを作成
    2. Vector Search Builder をセットアップ
    3. 接続情報を設定

    Part 1: Vector Search を体験（15 分）¶

    実際に Vector Search を動かして、どんなことができるか体験します。

    やること:

    1. 接続テストを実行
    2. デモアプリケーションを起動
    3. 「意味で検索」を体験

    学べること:

    • Vector Search の仕組み
    • 従来の検索との違い
    • 実際の活用シーン

    Part 2: IBM Bob で機能を追加（20 分）¶

    IBM Bob を使って、アプリケーションに新しい機能を追加します。

    やること:

    1. IBM Bob に日本語で指示を出す
    2. 検索結果に画像を表示する機能を追加
    3. 価格でフィルターする機能を追加

    学べること:

    • IBM Bob の使い方
    • AI アシスタントとの対話方法
    • 開発の効率化

    Part 3: 動作確認（10 分）¶

    追加した機能が正しく動作するか確認します。

    やること:

    1. 追加した機能をテスト
    2. IBM Bob のコードレビュー機能を使う
    3. 改善点を確認

    必要なもの¶

    必須¶

    • パソコン（Mac、Windows）
    • メモリ: 最低 4GB、推奨 8GB 以上

    • インターネット接続: 必須

    • IBM Bob

    • 既にインストールされ、使用できる状態

    • Web ブラウザ

    • Chrome、Firefox、Safari、Edge など

    講師から配布されるもの¶

    • Vector Search Builder（zip ファイル）
    • 接続情報（Milvus 接続情報）

    対象者¶

    こんな方におすすめ¶

    • 営業・セールスの方: 顧客に Vector Search をデモしたい
    • IBM Bob 初心者: IBM Bob を初めて使う
    • プログラミング初心者: コードを書いたことがない

    前提知識¶

    不要なもの:

    • プログラミング経験
    • データベースの知識
    • AI の専門知識

    あると良いもの:

    • パソコンの基本操作（ファイルのコピー、フォルダの作成など）
    • Web ブラウザの使い方

    期待される成果¶

    このハンズオンを完了すると、以下ができるようになります：

    理解できること¶

    • Vector Search とは何か
    • 従来の検索との違い
    • IBM Bob の基本的な使い方

    できるようになること¶

    • Vector Search のデモができる
    • IBM Bob で簡単な機能追加ができる
    • 顧客に Vector Search の価値を説明できる

    使用する技術¶

    Milvus（ミルバス）¶

    • 役割: ベクトルデータベース
    • 機能: ベクトルデータを保存・検索
    • 特徴: 高速・大規模データに対応

    埋め込みモデル（Hugging Face Transformers）¶

    • 役割: AI 埋め込みモデル
    • 機能: テキストをベクトル（数値の配列）に変換
    • 特徴: 日本語対応・高精度

    IBM Bob¶

    • 役割: AI 開発アシスタント
    • 機能: コード生成・レビュー・サポート
    • 特徴: 自然言語で指示できる
