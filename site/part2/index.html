    Part 2: IBM Bob で機能を追加しよう¶

    所要時間: 20 分

    このパートでは、IBM Bob を使って、Vector Search アプリケーションに新しい機能を追加します。

    このパートのゴール¶

    • IBM Bob の使い方を学ぶ
    • 日本語で指示を出してコードを生成する
    • 3 つの新機能を追加する

    IBM Bob とは？（復習）¶

    IBM Bob = AI がコーディングをサポートしてくれる開発ツール

    できること:

    • 日本語で「こういう機能が欲しい」と伝える
    • IBM Bob が自動的にコードを書いてくれる
    • コードの説明もしてくれる

    メリット:

    • プログラミング経験不要
    • 開発時間が大幅に短縮
    • 高品質なコードが生成される

    追加する機能¶

    このパートでは、以下の 3 つの機能を追加します：

    1. 商品画像の表示（7 分）
    2. 価格フィルター（7 分）
    3. レコメンド理由の表示（6 分）

    機能 1: 商品画像の表示（7 分）¶

    なぜこの機能が必要？

    現在の検索結果は文字だけです。商品画像があると視覚的に分かりやすく、ユーザー体験が向上します。

    ステップ 1: IBM Bob を開く¶

    IBM Bob の画面下部のチャット入力欄をクリック

    ステップ 2: Code モードに切り替え¶

    1. 画面右下の「Mode」セレクターをクリック
    2. 「Code」を選択

    Code モード = コードを書くための専用モード

    ステップ 3: IBM Bob に指示を出す¶

    チャット入力欄に以下を入力して Enter：

    検索結果に商品画像を表示してください

    ポイント:

    • 日本語で自然に書く
    • 具体的に何をしたいか伝える

    ステップ 4: IBM Bob の応答を待つ¶

    IBM Bob が自動的に以下を行います：

    1. 指示を理解
    2. 関連ファイルを探す
    3. コードを生成
    4. 説明を表示

    所要時間: 10〜30 秒程度

    ステップ 5: IBM Bob の提案を確認¶

    IBM Bob が以下のような提案をします：

    検索結果に商品画像を表示する機能を追加しました。

変更内容:
- app.py: レスポンスに image_url フィールドを追加
- 商品データから画像 URL を取得
- 検索結果に画像 URL を含める

    ステップ 6: 変更を承認¶

    1. IBM Bob の提案を読む
    2. 「承認」または「Apply」ボタンをクリック
    3. ファイルが自動的に保存される

    ステップ 7: 動作確認¶

    1. Swagger UI を開く（http://localhost:8000/docs）

    2. アプリケーションを再起動

    3. ターミナルで Ctrl+C（停止）

    4. python app.py（再起動）
    5. 検索を実行：
    {
  "query": "赤いスニーカー"
}
    1. 結果を確認：
    {
  "results": [
    {
      "product_name": "赤いランニングシューズ",
      "image_url": "https://example.com/images/red-shoes.jpg",
      "similarity_score": 0.92,
      "price": 8900
    }
  ]
}

    確認ポイント: image_urlフィールドが追加されている

    機能 1 完了チェック¶

    • IBM Bob に指示を出せた
    • IBM Bob がコードを生成した
    • 変更を承認した
    • 検索結果に image_urlが表示される

    機能 2: 価格フィルター（7 分）¶

    なぜこの機能が必要？¶

    価格帯で絞り込めると、予算に合った商品を探せます。

    ステップ 1: IBM Bob に指示を出す¶

    チャット入力欄に以下を入力して Enter：

    価格帯でフィルターできる機能を追加してください。
最小価格と最大価格を指定できるようにしてください。

    ステップ 2: IBM Bob の提案を確認¶

    IBM Bob が以下のような提案をします：

    価格フィルター機能を追加しました。

変更内容:
- app.py: min_price と max_price パラメータを追加
- 検索結果を価格帯でフィルタリング

    ステップ 3: 変更を承認¶

    「承認」または「Apply」ボタンをクリック

    ステップ 4: 動作確認¶

    1. アプリケーションを再起動
    2. 検索を実行：
    {
  "query": "スニーカー",
  "min_price": 5000,
  "max_price": 10000
}
    1. 結果を確認：5000 円〜10000 円の商品のみ表示される

    機能 2 完了チェック¶

    • IBM Bob に指示を出せた
    • 変更を承認した
    • 価格フィルターが動作する

    機能 3: レコメンド理由の表示（6 分）¶

    なぜこの機能が必要？¶

    なぜその商品がおすすめなのかを表示すると、ユーザーの納得感が高まります。

    ステップ 1: IBM Bob に指示を出す¶

    チャット入力欄に以下を入力して Enter：

    検索結果に「なぜこの商品がおすすめなのか」という理由を表示してください。
類似度スコアに基づいて理由を生成してください。

    ステップ 2: IBM Bob の提案を確認¶

    IBM Bob が以下のような提案をします：

    レコメンド理由表示機能を追加しました。

変更内容:
- app.py: recommendation_reason フィールドを追加
- 類似度スコアに基づいて理由を生成

    ステップ 3: 変更を承認¶

    「承認」または「Apply」ボタンをクリック

    ステップ 4: 動作確認¶

    1. アプリケーションを再起動
    2. 検索を実行：
    {
  "query": "初心者向けのカメラ"
}
    1. 結果を確認：
    {
  "results": [
    {
      "product_name": "入門用デジタルカメラ",
      "similarity_score": 0.95,
      "recommendation_reason": "検索内容と非常に高い類似性があります（95%）"
    }
  ]
}

    機能 3 完了チェック¶

    • IBM Bob に指示を出せた
    • 変更を承認した
    • レコメンド理由が表示される

    Part 2 完了チェック¶

    • 商品画像表示機能を追加した
    • 価格フィルター機能を追加した
    • レコメンド理由表示機能を追加した
    • すべての機能が正しく動作する

    Part 2 で学んだこと¶

    理解したこと¶

    • IBM Bob は日本語で指示を出せる
    • IBM Bob が自動的にコードを生成する
    • 変更内容を確認してから承認できる
    • プログラミング経験がなくても機能追加できる

    できるようになったこと¶

    • IBM Bob への指示の出し方
    • コード変更の承認方法
    • 機能追加の流れ
    • 動作確認の方法

    IBM Bob の便利な機能¶

    1. コードレビュー¶

    IBM Bob に「このコードをレビューして」と依頼すると、改善点を指摘してくれます。

    2. エラー解決¶

    エラーメッセージをコピーして「このエラーを解決して」と依頼すると、解決方法を提案してくれます。

    3. コード説明¶

    「このコードを説明して」と依頼すると、分かりやすく説明してくれます。

    次のステップ¶

    Part 2 が完了したら、Part 3: 動作確認に進みましょう！

    Part 3 では、追加した機能が正しく動作するか、詳しく確認します。

    困ったときは¶

    よくある質問¶

    Q1: IBM Bob が応答しない¶

    対処法:

    1. インターネット接続を確認
    2. IBM Bob を再起動
    3. 講師に相談

    Q2: 変更が反映されない¶

    対処法:

    1. ファイルが保存されているか確認
    2. アプリケーションを再起動
    3. ブラウザをリロード

    Q3: エラーが表示される¶

    対処法:

    1. エラーメッセージをコピー
    2. IBM Bob に「このエラーを解決して」と依頼
    3. 講師に相談
