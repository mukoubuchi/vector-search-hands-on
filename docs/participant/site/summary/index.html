    📚 学んだこと

    このハンズオンを通じて、以下のスキルと知識を習得しました：

    1. Building Blocks の実践的な活用

    • ✅ Vector Search の基本概念と仕組み
    • ✅ watsonx.data（Milvus）と watsonx.ai の連携
    • ✅ セマンティック検索の実装方法
    • ✅ Building Blocks を使った効率的な開発

    2. IBM Bob の活用方法

    • ✅ Code モードでの開発フロー
    • ✅ 自然言語による機能実装の依頼
    • ✅ コードレビュー機能の活用
    • ✅ AI アシスタントとの効果的なコミュニケーション

    3. Vector Search の実装

    • ✅ 検索結果の表示カスタマイズ
    • ✅ フィルター機能の追加
    • ✅ レコメンデーション理由の表示
    • ✅ Swagger UI を使った API テスト

    💼 業務での活用方法

    Sales チームの活用シーン

    1. 顧客へのデモ

    • 実践的なデモ: 実際に動作する Vector Search アプリケーションを使ってデモ
    • 直感的な説明: 「似た商品を探す」など、顧客に分かりやすい例で説明
    • IBM 製品の統合: watsonx.data と watsonx.ai の連携を実証

    2. 提案活動

    • ユースケースの提案: EC サイト、ナレッジベース、カスタマーサポートなど
    • ROI の説明: 開発効率化と検索精度向上による効果
    • 技術的な質問への対応: Building Blocks の実装を理解した上でエスカレーション

    3. 社内展開

    • ナレッジ共有: ハンズオン資料を社内で共有
    • フィードバック収集: 顧客の反応を Engineering チームにフィードバック

    Engineering チームの活用シーン

    1. 顧客プロジェクト

    • 迅速なプロトタイプ作成: Building Blocks を使った素早い実装
    • カスタマイズ: 顧客要件に合わせた機能追加
    • 品質保証: IBM Bob のコードレビュー機能を活用

    2. 社内開発

    • 開発効率化: IBM Bob を使った開発時間の短縮
    • ベストプラクティス: Building Blocks の実装パターンを参考
    • 技術検証: 新しい技術の迅速な検証
