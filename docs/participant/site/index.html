    IBM Bob × Building Blocks ハンズオンへようこそ 🎉

    IBM Bob を使って、Building Blocks の Vector Search（セマンティック検索）を体験する実践的なハンズオンです。

    🎯 ワークショップ概要

    Build Sales チームを対象とした、IBM Bob を使った Building Blocks ハンズオン

    このハンズオンでは、IBM Bob と Building Blocks を使いながら、Vector Search アプリケーションの開発プロセスを体験します。

    📚 学べること

    • ✅ Building Blocks を用いた実践的なハンズオン体験の提供
    • ✅ IBM Bob と Building Blocks の初学者向けの平易な内容で IBM Bob の活用方法を習得
    • ✅ SQL 知識不要で、直感的に理解できるユースケースの提供

    ⏱️ 所要時間

    合計: 約 60 分

    パート 内容 想定時間
    Part 1 環境確認とデモ 15 分
    Part 2 IBM Bob でカスタマイズ 30 分
    Part 3 動作確認とレビュー 15 分

    📋 事前準備チェックリスト

    ハンズオンを始める前に、以下の環境を準備してください：

    必須ソフトウェア

    • ✅ IBM Bob アカウント
    • 30日間無料トライアルから登録可能
    • ✅ Bob IDE - ダウンロードはこちら
    • ✅ Web ブラウザ - Chrome、Firefox、Safari、Edge 等

    推奨環境

    • 💻 PC/Mac/Linux/Windows - メモリ最低 4GB、推奨 8GB 以上
    • 🌐 インターネット接続 - 必須

    📝 実施内容

    テーマ

    IBM Bob × Building Blocks ハンズオン: Vector Search（セマンティック検索）

    対象 Building Block

    Vector Search — セマンティック検索とレコメンデーションシステム

    • IBM 製品: watsonx.data（Milvus）+ watsonx.ai + Docling
    • 機能: 意味ベースの検索、類似商品のレコメンデーション、RAG ベースの Q&A

    🚀 はじめましょう

    準備ができたら、事前準備のページに進んでください。

    💡 ワークショップのポイント

    実践的な学習体験

    このハンズオンは、単なる機能紹介ではありません。実際の開発シナリオに基づいた、実践的な学習体験を提供します。

    • 直感的なユースケース: 「似た商品を探す」など、顧客に説明しやすい機能を体験
    • SQL 不要: データベース知識がなくても使える自然言語検索
    • 開発効率の実感: IBM Bob による開発効率化を体感
    • IBM 製品の統合: watsonx.data と watsonx.ai のシームレスな連携を実証
