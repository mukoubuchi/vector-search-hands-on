## このハンズオンで学べること

### 1. Vector Searchとは？

**Vector Search（ベクトル検索）** は、言葉の「意味」を理解して検索する技術です。

#### 従来の検索との違い

**従来のキーワード検索**:
- 「赤いスニーカー」と検索 → 「赤い」と「スニーカー」という**文字**が含まれる商品を探す
- 「赤色のランニングシューズ」は見つからない（文字が違うため）

**Vector Search（意味で検索）**:
- 「赤いスニーカー」と検索 → 「赤い」「スニーカー」の**意味**を理解
- 「赤色のランニングシューズ」も見つかる（意味が似ているため）
- 「初心者向けカメラ」→「入門用デジタルカメラ」も見つかる

#### 実際の活用例

- **ECサイト**: 「似た商品を探す」機能
- **社内検索**: 「この資料に似た文書を探す」
- **カスタマーサポート**: 「似た質問を探す」

### 2. IBM Bobとは？

**IBM Bob** は、AIがコーディングをサポートしてくれる開発ツールです。

#### IBM Bobでできること

- **自然な日本語で指示**: 「検索結果に画像を表示して」と伝えるだけ
- **コードを自動生成**: IBM Bobがコードを書いてくれる
- **コードレビュー**: 書いたコードの問題点を指摘してくれる

#### 従来の開発との違い

**従来の開発**:
1. プログラミング言語を学ぶ（数ヶ月〜数年）
2. コードを書く（数時間〜数日）
3. バグを修正する（数時間〜数日）
4. テストする（数時間）

**IBM Bobを使った開発**:
1. 日本語で「こういう機能が欲しい」と伝える（数分）
2. IBM Bobがコードを生成（数秒）
3. 動作確認（数分）
4. 必要に応じて修正を依頼（数分）

## 参考情報

### Building Blocksとは？

**Building Blocks** は、IBMが提供する再利用可能なコンポーネント集です。

#### Building Blocksの特徴

**目的**:
- よくある機能を簡単に実装
- 開発時間を大幅に短縮
- ベストプラクティスを組み込み済み

**提供されるBuilding Blocks**:
- **Vector Search**: 意味で検索する機能（このハンズオンで使用）
- **Text2SQL**: 自然言語でデータベースを検索
- **Agent Builder**: AIエージェントを構築
- **Document Processing**: 文書を自動処理
- その他多数

**メリット**:
- ゼロから作るより圧倒的に速い（数日→数時間）
- 品質が保証されている（IBMが検証済み）
- カスタマイズ可能（自社の要件に合わせて調整）
- 保守が容易（IBMがアップデート提供）

#### Building Blocksの使い方

1. **選択**: 必要な機能のBuilding Blockを選ぶ
2. **設定**: 環境に合わせて設定を調整
3. **統合**: 既存システムに組み込む
4. **カスタマイズ**: IBM Bobで機能を追加・修正

#### 実際の活用例

**ECサイト**:
- Vector Search Building Block → 「似た商品を探す」機能
- Recommendation Building Block → 「あなたへのおすすめ」機能

**社内システム**:
- Text2SQL Building Block → 「売上データを自然言語で検索」
- Document Processing Building Block → 「契約書を自動分類」

**カスタマーサポート**:
- Agent Builder Building Block → 「AIチャットボット」
- Vector Search Building Block → 「似た質問を検索」

### 使用する技術

このハンズオンでは、以下の技術を使用します：

#### Milvus（ミルバス）

- **役割**: ベクトルデータベース
- **機能**: ベクトルデータを保存・検索
- **特徴**: 高速・大規模データに対応
#### 埋め込みモデル（Hugging Face Transformers）


- **役割**: AI埋め込みモデル
- **機能**: テキストをベクトル（数値の配列）に変換
- **特徴**: 日本語対応・高精度

#### IBM Bob

- **役割**: AI開発アシスタント
- **機能**: コード生成・レビュー・サポート
- **特徴**: 自然言語で指示できる
