# Part 2: IBM Bobで機能を追加しよう

**所要時間**: 20分

このパートでは、IBM Bobを使って、Vector Searchアプリケーションに新しい機能を追加します。

## このパートのゴール

- IBM Bobの使い方を学ぶ
- 日本語で指示を出してコードを生成する
- 3つの新機能を追加する

## IBM Bobとは？（復習）

**IBM Bob** = AIがコーディングをサポートしてくれる開発ツール

**できること**:

- 日本語で「こういう機能が欲しい」と伝える
- IBM Bobが自動的にコードを書いてくれる
- コードの説明もしてくれる

**メリット**:

- プログラミング経験不要
- 開発時間が大幅に短縮
- 高品質なコードが生成される

## 追加する機能

このパートでは、以下の3つの機能を追加します：

1. **商品画像の表示**（7分）
2. **価格フィルター**（7分）
3. **レコメンド理由の表示**（6分）

## 機能1: 商品画像の表示（7分）

!!! info "なぜこの機能が必要？"
    現在の検索結果は文字だけです。商品画像があると視覚的に分かりやすく、ユーザー体験が向上します。

### ステップ1: IBM Bobを開く

IBM Bobの画面下部のチャット入力欄をクリック

### ステップ2: Codeモードに切り替え

1. 画面右下の「Mode」セレクターをクリック
2. 「Code」を選択

**Codeモード** = コードを書くための専用モード

### ステップ3: IBM Bobに指示を出す

チャット入力欄に以下を入力してEnter：

```
検索結果に商品画像を表示してください
```

**ポイント**:

- 日本語で自然に書く
- 具体的に何をしたいか伝える

### ステップ4: IBM Bobの応答を待つ

IBM Bobが自動的に以下を行います：

1. 指示を理解
2. 関連ファイルを探す
3. コードを生成
4. 説明を表示

**所要時間**: 10〜30秒程度

### ステップ5: IBM Bobの提案を確認

IBM Bobが以下のような提案をします：

```
検索結果に商品画像を表示する機能を追加しました。

変更内容:
- app.py: レスポンスにimage_urlフィールドを追加
- 商品データから画像URLを取得
- 検索結果に画像URLを含める
```

### ステップ6: 変更を承認

1. IBM Bobの提案を読む
2. 「承認」または「Apply」ボタンをクリック
3. ファイルが自動的に保存される

### ステップ7: 動作確認

1. Swagger UIを開く（`http://localhost:8000/docs`）
2. アプリケーションを再起動
   - ターミナルで ++ctrl+c++（停止）
   - `python app.py`（再起動）
3. 検索を実行：

   ```json
   {
     "query": "赤いスニーカー"
   }
   ```

4. 結果を確認：

   ```json
   {
     "results": [
       {
         "product_name": "赤いランニングシューズ",
         "image_url": "https://example.com/images/red-shoes.jpg",
         "similarity_score": 0.92,
         "price": 8900
       }
     ]
   }
   ```

**確認ポイント**: `image_url`フィールドが追加されている

- [ ] IBM Bobがコードを生成した
- [ ] 変更を承認した
- [ ] 検索結果に`image_url`が表示される

## 機能2: 価格フィルター（7分）

### なぜこの機能が必要？

価格帯で絞り込めると、予算に合った商品を探せます。

### ステップ1: IBM Bobに指示を出す

チャット入力欄に以下を入力してEnter：

```
価格帯でフィルターできる機能を追加してください。
最小価格と最大価格を指定できるようにしてください。
```

### ステップ2: IBM Bobの提案を確認

IBM Bobが以下のような提案をします：

```
価格フィルター機能を追加しました。

変更内容:
- app.py: min_priceとmax_priceパラメータを追加
- 検索結果を価格帯でフィルタリング
```

### ステップ3: 変更を承認

「承認」または「Apply」ボタンをクリック

### ステップ4: 動作確認

1. アプリケーションを再起動
2. 検索を実行：

   ```json
   {
     "query": "スニーカー",
     "min_price": 5000,
     "max_price": 10000
   }
   ```

3. 結果を確認：5000円〜10000円の商品のみ表示される

- [ ] IBM Bobに指示を出せた
- [ ] 変更を承認した

### ステップ3: 変更を承認

## IBM Bobの便利な機能

### 1. コードレビュー

IBM Bobに「このコードをレビューして」と依頼すると、改善点を指摘してくれます。

### 2. エラー解決

エラーメッセージをコピーして「このエラーを解決して」と依頼すると、解決方法を提案してくれます。

### 3. コード説明

「このコードを説明して」と依頼すると、分かりやすく説明してくれます。

## 困ったときは

### よくある質問

#### Q1: IBM Bobが応答しない

**対処法**:
1. インターネット接続を確認
2. IBM Bobを再起動
3. 講師に相談

#### Q2: 変更が反映されない

**対処法**:
1. ファイルが保存されているか確認
2. アプリケーションを再起動
3. ブラウザをリロード

#### Q3: エラーが表示される

**対処法**:
1. エラーメッセージをコピー
2. IBM Bobに「このエラーを解決して」と依頼
3. 講師に相談
