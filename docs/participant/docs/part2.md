**所要時間**: 30 分

このパートでは、IBM Bob を使って Vector Search アプリケーションに機能を追加します。

## 1. 検索結果の表示カスタマイズ（10 分）

### 目的

検索結果をより分かりやすく表示するために、商品画像と類似度スコアを追加します。

### 実装手順

1. **IBM Bob の Code モードで実装開始**

   - IBM Bob IDE を開く
   - Code モードに切り替える

2. **IBM Bob に実装を依頼**

   以下のプロンプトを IBM Bob に入力します：

   ```text
   検索結果に商品画像と類似度スコアを追加してください。
   ```

3. **生成されたコードの確認**

   IBM Bob が生成したコードを確認し、以下の点を理解します：

   - 画像 URL の取得方法
   - 類似度スコアの表示形式
   - レスポンスの構造

### 期待される結果

- 検索結果に商品画像が表示される
- 各結果に類似度スコア（0.0〜1.0）が表示される
- より視覚的に分かりやすい検索結果

## 2. フィルター機能の追加（10 分）

### 目的

価格帯とカテゴリでフィルターできる機能を追加し、より精度の高い検索を実現します。

### 実装手順

1. **IBM Bob に実装を依頼**

   以下のプロンプトを IBM Bob に入力します：

   ```text
   価格帯とカテゴリでフィルターできる機能を追加してください。
   ```

2. **フィルターと Vector Search の組み合わせ**

   IBM Bob が提案する実装を確認し、以下を理解します：

   - フィルター条件の指定方法
   - Vector Search とフィルターの組み合わせ方
   - パフォーマンスへの影響

### 期待される結果

- 価格帯でフィルター可能（例: 1000 円〜5000 円）
- カテゴリでフィルター可能（例: スニーカー、ノートパソコン）
- フィルター条件とセマンティック検索の組み合わせ

### 実装例

```python
# フィルター条件の例
filters = {
    "price_range": {"min": 1000, "max": 5000},
    "category": "スニーカー"
}

# Vector Search + フィルター
results = search_with_filters(query="赤い", filters=filters)
```

### 実装手順

   以下のプロンプトを IBM Bob に入力します：


2. **段階的な実装**

   IBM Bob の提案を確認しながら、以下を段階的に実装します：

   - 類似特徴の抽出

### 期待される結果

- 類似している特徴が明確に示される
