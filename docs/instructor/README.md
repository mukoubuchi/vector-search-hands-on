# 講師向けドキュメント

このディレクトリには、Vector Search ハンズオンの講師向けドキュメントが含まれています。

## ドキュメント一覧

### [環境セットアップガイド](setup-guide.md)

ハンズオンの環境準備に関する詳細なガイドです。

**内容:**
- Docker / Docker Compose のインストール
- Milvus 環境の起動
- watsonx.ai の設定
- 受講者への接続情報の共有方法
- トラブルシューティング

**対象:**
- ハンズオンを実施する講師
- 環境を準備する担当者

## 推奨セットアップパターン

**パターンA: 講師が共有環境を提供（推奨）**
- 講師のPCでMilvusを起動
- 受講者はIBM Bobのみインストール
- 5-10人程度の小規模ハンズオンに最適

**メリット:**
- ✅ 受講者の準備が最小限
- ✅ 全員が同じ環境を使用
- ✅ トラブルシューティングが容易

**デメリット:**
- ⚠️ 講師のPCに負荷が集中
- ⚠️ ネットワーク接続が必要
